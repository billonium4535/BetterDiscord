To update project on server:
git pull origin "First-Stable-Release-(v2)"

To clone project on server:
git clone https://github.com/username/repository-name.git
e.g. 'git clone -b "First-Stable-Release-(v2)" https://github.com/billonium4535/BetterDiscord.git'

To run server:
python3 server.py
